# Documenter Documentation

Welcome to the documentation for Documenter.

## Overview
Documenter is a local GUI tool for managing project documentation with Gemini 3 and GitHub integration. NYX is a AI content studio which will be helpful in Content Creation.

## Features
- **Local File Sync**: Reads .md files.
- **AI Chat Interface**: Chat with your docs.
- **Agentic File Modification**: Updates files via AI.
- **GitHub Auto-Commit**: Syncs changes to GitHub.
