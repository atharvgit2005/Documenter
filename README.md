# Documenter

A local GUI tool for managing project documentation with Gemini 3 and GitHub integration.

## Getting Started

1.  Run `npm install`
2.  Run `npm run dev`
3.  Open [http://localhost:3000](http://localhost:3000)

## Features

-   **Stack**: Next.js, Tailwind CSS, shadcn/ui.
-   **AI**: Gemini 3 Pro via `@google/generative-ai`.
-   **Sync**: Local file system + GitHub (Octokit).

## Documentation

Documentation is located in the `/docs` folder.
